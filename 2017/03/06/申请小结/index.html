申请季有感

One Punch CAT
2017-03-06
总结, 申请

从开始着手准备申请美帝开始，也就是去年的3月，到现在申请季的尾声，这一年有太多可以说的故事。生活和学习上的习惯和心态，对自己的定位，以及对未来的思考，我都发生了不小的改变。

申请的第一阶段，必然是准备GRE和托福。然而，从三月开始，一直到暑期结束，我都处于全职实习的状态。特别是在MSRA的实习，因为项目实在太有趣，根本分不出心思去想别的。所以我备考GRE和托福的策略便是短时间大量的集中训练。我在从暑期回校（十月初）到十一月十一号，这差不多一个月的时间，参加了一次托福和两次GRE。现在回顾，也真不知道自己是如何走过的：一个月里，差不多背了将近八千的单词，做了数不尽的阅读，填空。还要在不尽人意的模考分数中给自己鼓励。特别记得，第一次GRE失利了，走出考场的时候，虽然没有哭，但也自暴自弃地想，这样的日子我受够了，再也不想继续了，就拿着这分数罢了吧。然后回家咬着牙立刻定了三星期后的考位。最后虽然不是得到了一个逆天的成绩，但也对得起这一个月的坚持。105 + 325的标准配置，十一月十一号，光棍节，我和托福，GRE双双分手。

除了坚持刷题背单词，我每天还一定都会花四十五分的时间，不是去长跑，就是去游泳。一方面是为了清醒一下自己疲劳过度的脑袋，另一方面是好好沉淀一下浮躁的心情。准备语言考试，真的没有捷径。多少的输入，就是多少的输出。依赖所谓的鸡精，总归是要还的。即使不是在考试分数上，也会在日后留学的日子里。

除开 PS 和 CV，最关键的就是选校。因为对CS较感兴趣，自己在班上一直能维持着高GPA，也就是所谓的各科全A(4.0)的成绩。然而，身在小本，好的成绩往往有时候让我错误定位自己，一开始竟只自信地把选校的名单全部集中在了专排前十的CS项目。然而在仔细研究了往届录取的数据之后，我首先发现我的背景实则处于一个尴尬的地位。高GPA，然而本科牌子不够响。英语不算太低，但也仅仅是过线。实习背景丰富，但是项目却杂，并且没有科研。其次，除开general的CS项目，各个学校有很多不错的professional的项目，这些可能更合适我的背景。

最后，我选择只申请少数几个，我一定有兴趣去的项目，并且采用一些曲线救国的策略。

  • 彩票：

    • CMU - MCDS (System Track): 这个项目完美契合我对于硕士的课程的需求，而且是我心心念念最想去的学校，但是因为本科以及英语分不高这一点，并不是抱有太大的希望。
    • Stanford: 申请CS大家都会象征性买的彩票
    • UCLA: 卡本科的学校，天气好，位置好，而且load并没有CMU大然而就业是差不多的。

  • 主申:

    • CMU - INI MSIN: 这个项目的选课如果得当可以曲线救国当成MCDS的System来上，录取的bar也比MCDS低。而且还是梦校CMU中极大机率可以拿到小奖的项目。
    • UIUC - MCS: 玉米地的professional cs专业，不知道为什么就是很喜欢这个学校 ;)

  • 保底:

    • USC: 结合去年我们学校的数据，USC算是比较稳妥的保底学校

申请环节很是繁琐，为了方便统计，我便做了一个表格，把各个项目需要的材料和截止日期什么的都列在了一起。然后就是写了一分大致的PS并且修改成每个学校的版本。在截止日期前提交之后，就开始漫长地等待。

等待的时间比准备的时间，甚至是准备GT的那一段时间更需要一颗强大的心脏。有一种未来并不是捏在自己手里，尽人事听天命的感觉。这种心情在二月底还没有收到任何一家学校的消息，并且得知MCDS已经发了第一波的AD时到了最大化，以至于自己都没有心情好好集中在学校的课业上。直到3月4号，收到了CMU-MSIN的录取，心里的石头才真正的放下。

虽然这一路自己走的不容易，但是有一些苦难确实是可以避免的。因为犹豫申美还是留英，以及拖延症，导致GT的高度紧张以及最后只是刚刚过线的结果。在等AD的时间里，因为内心的弱小，也不知道浪费了多少时间在毫无意义的担心中。说到底自己还是把申请看的太“重”，而忘记学校的录取，只是开始，远远不是结束。

如果彩票档的AD不来，大概率就会从了MSIN这个项目。据说这个项目是CMU出了名的累，有很多系统方向的课，并且每个课都有很大的作业量。有人说，去美国找工作，学校只是敲门砖，去CMU累成狗但是最后找到的工作可能和比较轻松的项目不会有太大差距，所以又是何苦呢。这个观点确实挺对，然而对真的很喜欢技术的人来说，找工作只是最低标准，不要浪费求学的时间，学到最多的干货，将来才不会后悔。不忘初心，方得始终。

感谢所有在申请季帮我的，鼓励我的，以及给我reference的大家。不论最后能收到几个AD，我已经觉得圆满了。
