安卓小记-基础动画的使用

One Punch CAT
2015-02-07
安卓

动画效果有时候可以让一个应用瞬间活泼起来。正好在做project的时候想使用动画为UI加一点分，所以这边就来稍微简单的记录一下最基础的动画。
安卓的Animation可以有JavaCode和XML定义：

XMl - 存放于res/anim文件夹中

  • alpha 渐变透明度动画效果
  • scale 渐变尺寸伸缩动画效果
  • translate 画面转换位置移动动画效果
  • rotate 画面转移旋转动画效果

Java code

  • AlphaAnimation 渐变透明度动画效果
  • ScaleAnimation 渐变尺寸伸缩动画效果
  • TranslateAnimation 画面转换位置移动动画效果
  • RotateAnimation 画面转移旋转动画效果

在XML中定义动画

这里有每一种Aminate具体工作时需要定义，以及可以定义的一些标签
其中有一些标签比较值得注意。

  • interpolator - 定义了动画的插补器，可以控制动画进行的速率。以下是安卓自带的
    • @android:anim/accelerate_interpolator： 越来越快
    • @android:anim/decelerate_interpolator：越来越慢
    • @android:anim/accelerate_decelerate_interpolator：先快后慢
    • @android:anim/anticipate_interpolator: 先后退一小步然后向前加速
    • @android:anim/overshoot_interpolator：快速到达终点超出一小步然后回到终点
    • @android:anim/anticipate_overshoot_interpolator：到达终点超出一小步然后回到终点
    • @android:anim/bounce_interpolator：到达终点产生弹球效果，弹几下回到终点
    • @android:anim/linear_interpolator：均匀速度。
  • startOffset - 定义了动画的延迟
  • duration - 动画的持续时间
  • repeat - 动画的循环方式
  • fillAfter - 当设置为true, 动画结束后停留在最后一帧
  • fillBefore - 当设置为true, 动画结束后停留在地一帧
  • RepeatMode - 定义动画的循环模式，一般有REVERSE和RESTART两种
  • RepeatCount - 定义动画的循环次数, 可以设置为INIFITE
