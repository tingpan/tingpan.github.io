安卓小记-Activity的生命周期

One Punch CAT
2015-02-12
安卓

关于Activity的生命周期，其实就是每一个Activity在什么时候被创造，什么时候被销毁，又是在什么时候调用什么样的方法。首先，下图其实已经很明确的说明了安卓的生命周期

可以看到，每一个安卓Activity都有共同的一些方法，并且这些方法都是可以被重写的。所以如果要知道如何重写合适的方法，就要知道在什么时候系统会调用Activity的哪一些方法。

方法的调用

  • 在Activity被创建时：
    • onCreate(); -> onStart(); -> onResume();
  • 在按下Activity被置于后层再用Back返回时
    • onPause(); -> onResume();
  • 在按下Home键再回来时
    • onPause(); -> onStop(); -> onRestart(); -> onStart(); -> onResume();
  • 在Activity被销毁时
    • onPause(); -> onStop(); -> onDestroy();

一些需要注意的问题

  • 如果想让程序在按下home键回来之后记录用户的操作（如textinput中存在的文字）需要重写一下onPaust(); 和 onRestart();方法
  • 在绑定了receiver时要记得在onPause();或者Activity销毁时解绑，否则再次创建Activity时会产生错误。
